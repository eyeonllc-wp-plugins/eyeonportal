# EyeOn Portal — WordPress Plugin

Connect your WordPress site to [EyeOn Portal](https://eyeonportal.com/) and display live center content with Elementor widgets.

**[Download WordPress Plugin](https://github.com/mycenterportal/wp-eyeonportal-plugin/releases/latest/download/eyeonportal.zip)**

## What it does

Center content — stores, deals, events, news, careers, banners, hours, and more — is managed in EyeOn Portal. This plugin pulls that data via an API access token and renders it on your public WordPress site. The environment (production, staging, or local dev) is detected automatically from the token prefix.

## Elementor widgets

Add widgets from the **EyeOn Portal** category in Elementor:

- **Stores** — retailer directory with categories, tags, and detail pages
- **Deals** — promotions and offers listing
- **Events** — calendar and event listings
- **Careers** — job listings
- **News** — blog and news articles
- **Banners** — homepage slider from EyeOn Portal
- **Center Hours** — opening hours display
- **Map** — interactive 3D floor map
- **Search** — site-wide center content search

## Other features

- **Single detail pages** — dedicated URLs for individual stores, deals, events, careers, and news articles
- **AI chatbot** — optional on-site assistant when enabled for your center in EyeOn Portal
- **SEO / Open Graph** — meta tags for single pages, with Rank Math integration and fallbacks
- **Auto-updates** — plugin checks GitHub releases and updates from the WordPress admin
- **Remote fleet updates** — EyeOn Portal staff can push plugin updates to registered sites via Manage WP

## Requirements

- WordPress 6.0+
- [Elementor](https://wordpress.org/plugins/elementor/)
- API access token from EyeOn Portal (generated by a super admin)

## Quick setup

1. Download the zip and install it into `wp-content/plugins/eyeonportal`
2. Activate the plugin in WordPress
3. Go to **EyeOn Portal → Settings** and paste your API access token
4. Assign listing pages (Stores, Deals, Events, News, Careers, Map) in plugin settings
5. Edit pages in Elementor and add EyeOn Portal widgets

Manage all center content at [eyeonportal.com](https://eyeonportal.com/).
